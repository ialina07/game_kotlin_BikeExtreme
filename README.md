# Кейс 1: создание десктоп приложения, позволяющего администрировать процесс игры в рамках одной партии

### Cистема для настольной игры «Велоэкстрим».

### Выполнила: Иванова Алина


## О проекте

Это приложение для администрирования партий.  
Ведущий записывает ходы игроков, система проверяет их корректность по правилам, сохраняет историю и считает статистику.

### Игра «Велоэкстрим» (BikeExtreme)

Гонка на велосипедах с динамической погодой.  
Игроки управляют ресурсами: энергия, состояние велосипеда, вода.  
Побеждает тот, кто первым достигнет 30-й клетки.

## Функции системы

- Запись ходов игроков
- Валидация ходов по правилам игры
- Хранение истории партий
- Расчёт статистики и рейтингов игроков
